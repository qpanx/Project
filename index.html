How It Works

Detection

Five IR sensors mounted at the front constantly scan the surface contrast to differentiate between the black line and the white floor.

Processing

The Arduino reads the digital inputs from the sensors. Based on which sensor is triggered, it calculates the error correction needed.

Actuation

The L298N motor driver receives PWM signals to adjust the speed of the left and right yellow gear motors to steer the chassis.

Hardware Components

The build features a customized double-decker acrylic chassis. The wiring is managed via a breadboard on the top deck, connecting the 5-sensor array to the main controller.

Arduino Logic

The core algorithm handling the line detection.

// Sensor Pins
#define S1 2
#define S2 3
#define S3 4 // Middle Sensor
#define S4 5
#define S5 6

void setup() {
  pinMode(S1, INPUT);
  pinMode(S2, INPUT);
  pinMode(S3, INPUT);
  pinMode(S4, INPUT);
  pinMode(S5, INPUT);
  
  // Initialize Motors...
}

void loop() {
  // Read Sensors
  int s1_val = digitalRead(S1);
  int s2_val = digitalRead(S2);
  int s3_val = digitalRead(S3);
  int s4_val = digitalRead(S4);
  int s5_val = digitalRead(S5);

  // Logic: 0 = White, 1 = Black Line
  
  if (s3_val == 1) {
    moveForward(); // Center sensor sees line
  } 
  else if (s2_val == 1) {
    turnLeftSafe(); // Slightly left
  }
  else if (s4_val == 1) {
    turnRightSafe(); // Slightly right
  }
  else if (s1_val == 1) {
    turnLeftHard(); // Sharp left
  }
  else if (s5_val == 1) {
    turnRightHard(); // Sharp right
  }
  else {
    stopCar();
  }
}
